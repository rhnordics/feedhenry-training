# FeedHenry Training

This repository contains a series of labs covering core topics of mobile app development on the FeedHenry platform. The labs cover App Forms, Cloud Apps, Client Apps, Cloud Services and several mBaaS APIs by incrementally developing a sample mobile application called Triply. Labs 4-6 build upon each other and need to be followed in order.

* [Lab 1 - App Forms](lab-01)
* [Lab 2 - Cloud Apps](lab-02)
* [Lab 3 - Client Apps](lab-03)
* [Lab 4 - mBaaS API Database Storage](lab-04)
* [Lab 5 - mBaaS API Cloud](lab-05)
* [Lab 6 - mBaaS Services](lab-06)
